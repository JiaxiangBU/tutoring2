2.  文件夹构造的逻辑，按照人名完成，`R/` 是一些集成的函数，方便调用，避免重复代码。
3.  目前完成问题174个，如下。

The goal of tutoring2 is to collect the issues and related answers.
[tutoring](https://github.com/JiaxiangBU/tutoring) is depreciated on the
security grounds.
